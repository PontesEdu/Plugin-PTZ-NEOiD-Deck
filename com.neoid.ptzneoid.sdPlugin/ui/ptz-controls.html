PTZ NEOiD - Controls PTZ

Para usar os controles PTZ, primeiro escolha a direção que deseja mover a câmera, como Home, Cima, Baixo, Esquerda, Direita ou opções Diagonais. Em seguida, pressione o botão para controlar a câmera selecionada. Você também pode adicionar um IP de Câmera Padrão no campo de IP Default. Se nenhuma câmera estiver selecionada no menu Select Camera ou se estiver exibindo No Camera, os controles irão usar automaticamente o IP Padrão para operar a câmera que você adicionou.

To use the PTZ controls, first choose the direction you want the camera to move, such as Home, Up, Down, Left, Right, or Diagonal options. Then press the button to control the selected camera. You can also enter a Default Camera IP in the Default IP field. If no camera is selected in the Select Camera menu or if it shows No Camera, the controls will automatically use the Default IP to operate the camera you added.
