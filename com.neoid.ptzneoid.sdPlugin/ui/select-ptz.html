PTZ NEOiD - Select PTZ camera

Para controlar uma câmera PTZ, primeiro você precisa registrá-la inserindo o endereço IP no campo “Camera IP” e, se quiser, atribuindo um nome no campo “name”. Caso a câmera adicionada não esteja conectada, o sistema exibirá automaticamente “Not connected” no lugar do nome. Sempre que for controlar uma câmera, certifique-se de selecioná-la aqui antes.

To control a PTZ camera, you must first register it by entering its IP address in the “Camera IP” field and optionally giving it a name in the “name” field. If the added camera is not connected, the system will automatically display “Not connected” instead of the name. Whenever you want to control a camera, make sure to select it here first.
